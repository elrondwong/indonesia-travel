### 非必要、可选或不建议

| 物品/做法 | 结论 | 理由 |
|---|---|---|
| 额外再买镜头 | 非必要 | 当前三支镜头覆盖 14-200mm，已足够完成全程主片；再加镜头只会增加重量和换镜风险。 |
| CPL | 可选增强 | 对瀑布、潮池、海水反光有帮助，但不是没有就不能拍；广角天空可能不均。 |
| ND8 / ND64 | 可选增强 | 瀑布和海浪慢门好用，但所有点都可以先拍快门版；不是刚性安全装备。 |
| L 型快装板 / 腕带 / 快挂 | 建议但非硬必需 | 对竖幅接片、悬崖和瀑布安全有帮助；没有也能拍，但要更谨慎。 |
| 自带全套浮潜面镜/呼吸管 | 可选 | 若船家装备干净可用，不一定要自带；但运动相机防水壳、浮力手柄、防丢绳更关键。 |
| 自己购买 Ijen 专业防毒面具 | 可选，必须确认向导提供 | 多数 Ijen 行程会提供 gas mask；若向导无法明确提供或规格太差，再考虑自带。 |
| 无人机 | 不建议作为本次刚需 | Bromo/Ijen/Penida 涉及国家公园、海崖、强风、人群和当地限制；本攻略不把无人机纳入必要装备。 |
| 主相机带上浮潜小船裸放 | 不建议 | 盐雾、浪、船晃和无人看管风险高；水下用运动相机，主相机只在安全岸上/大船上使用。 |
| 拖鞋走 Bromo / Ijen / Penida 东线 | 不建议 | 沙海、夜爬、楼梯和碎石路都需要包脚抓地鞋。 |
| 厚棉衣/牛仔裤/棉袜下瀑布或夜爬 | 不建议 | 湿后重、冷、干得慢；优先速干层 + 防风防雨外层。 |
| 大量现金和全部证件放同一个包 | 不建议 | 分开放，核心证件、卡、电池、存储卡随身。 |
| 全尺寸洗发水、沐浴露、护肤大瓶 | 非必要 | 酒店通常有基础洗护；自己带小分装就够，液体太多增加漏液和行李重量。 |

### 装备取舍总判断

```text
1. 已有相机和镜头够，不需要再补镜头。
2. 如果只能补最关键的缺口，优先级是：三脚架/电池/卡/备份 > 防水防尘 > 头灯保暖 > Ijen 防毒面具确认 > Penida 海上防晒防水。
3. 非必要装备不要喧宾夺主：CPL/ND/L 板/腕带属于提升成功率，不是安全底线。
4. 现场确认项和实物一样重要；Bromo、Ijen、Penida 的司机/向导/船家确认必须在出发前打勾。
```
