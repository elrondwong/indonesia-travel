这份文件把机票/高铁、酒店、包车/租车、门票从 Markdown 表格推进到结构化 CSV。等出发城市、日期、人数、房间数和预算确定后，先复制模板，再把每个候选填进去，最后用脚本检查是否漏了价格、退改、证据和状态。

## 使用顺序

```bash
cp data/gannan_flight_candidates.template.csv data/gannan_flight_candidates.csv
cp data/gannan_hotel_candidates.template.csv data/gannan_hotel_candidates.csv
cp data/gannan_transport_quotes.template.csv data/gannan_transport_quotes.csv
cp data/gannan_ticket_candidates.template.csv data/gannan_ticket_candidates.csv
```

检查全部候选表：

```bash
python3 scripts/check_gannan_order_candidates.py \
  data/gannan_flight_candidates.csv \
  data/gannan_hotel_candidates.csv \
  data/gannan_transport_quotes.csv \
  data/gannan_ticket_candidates.csv
```

生成可读复核报告：

```bash
python3 scripts/gannan_order_report.py
```

检查候选是否覆盖完整旅程：

```bash
python3 scripts/check_gannan_order_coverage.py --strict
```

只检查空模板是否可用：

```bash
python3 scripts/check_gannan_order_candidates.py
```

## 状态口径

| status | 含义 |
|---|---|
| candidate | 初始候选，还没做完整比较 |
| shortlist | 进入短名单，可继续谈价或准备下单 |
| booked | 已下单，必须有订单截图或确认记录 |
| watch | 暂时保留，等天气、开放、价格或联系人回复 |
| rejected | 淘汰，notes 写原因 |
| backup | 备选方案，用于主方案涨价/无房/关闭时切换 |

## 大交通候选表

模板：[data/gannan_flight_candidates.template.csv](data/gannan_flight_candidates.template.csv)

必填重点：

- `direction`：outbound / return / train / airport-transfer。
- `final_price_cny`：最终含税价，不填搜索列表低价。
- `baggage_or_luggage`：托运、随身、超重和摄影器材口径。
- `refund_change_policy`：退改规则，必须截图。
- `arrival_transfer`：到兰州后怎么到酒店、取车点或司机集合点。
- `evidence_file`：下单页或票规截图文件名。

淘汰规则：

- [ ] 晚到兰州且压缩 D1。
- [ ] 返程太早导致 D8 必须从阿坝县硬冲兰州。
- [ ] 无托运行李或退改太差，且价格没有明显优势。
- [ ] 同日相近时段高于其他候选 30% 以上。

## 酒店候选表

模板：[data/gannan_hotel_candidates.template.csv](data/gannan_hotel_candidates.template.csv)

必填重点：

- `stay_day`：D0-D8。
- `area`：兰州、夏河、合作、扎尕那、唐克/若尔盖、阿坝县、合作/临夏。
- `parking_confirmed`、`hot_water_heating_confirmed`、`early_departure_confirmed`：用 yes / no / unknown。
- `real_photo_checked`：是否看过真实房间、卫生间、窗景或停车位置照片。
- `cancel_deadline`：免费取消截止时间，没有免费取消就写 no-free-cancel。

淘汰规则：

- [ ] 扎尕那和阿坝县不能确认热水/供暖/停车。
- [ ] 早出门不方便，影响日出或莲宝叶则。
- [ ] 只发平台图，不发真实房间/窗景。
- [ ] 不可取消但价格并不明显便宜。

## 包车 / 租车候选表

模板：[data/gannan_transport_quotes.template.csv](data/gannan_transport_quotes.template.csv)

必填重点：

- `type`：charter / self-drive / agency。
- `total_price_cny`：总价，含不含油费、高速、停车必须分列。
- `driver_meals_lodging`：司机食宿谁承担。
- `insurance_confirmed`：商业险、座位险、租车保险、道路救援。
- `night_drive_policy`：是否拒绝危险夜路。
- `cancellation_policy`：定金和取消规则。

淘汰规则：

- [ ] 费用包含项不清楚。
- [ ] 主动建议赶夜路或不重视高反。
- [ ] 车型、座位、行李空间不匹配。
- [ ] 保险和救援说不清楚。

## 门票候选表

模板：[data/gannan_ticket_candidates.template.csv](data/gannan_ticket_candidates.template.csv)

必填重点：

- `scenic_area`：拉卜楞寺、甘加秘境、扎尕那、郎木寺、花湖、黄河九曲、莲宝叶则等。
- `target_date`：计划游览日期。
- `ticket_type`：成人票、观光车、组合票、扶梯、预约票等。
- `open_scope_confirmed`：开放范围是否确认，尤其扎尕那和莲宝叶则。
- `latest_entry` / `latest_exit`：影响日落和蓝调。
- `evidence_file`：官方/OTA/酒店/景区回复截图。

淘汰或观察规则：

- [ ] 扎尕那票价口径冲突但没有二次确认。
- [ ] 莲宝叶则只确认票价，没有确认开放范围和景区交通。
- [ ] 黄河九曲日落后下撤时间不清楚。
- [ ] 花湖观光车或栈道开放不明确。

## 最终比较方法

每类候选至少保留：

| 类型 | 候选数量 | 最终动作 |
|---|---:|---|
| 大交通 | 2-3 | 选一个下单，一个保留到出票前 |
| 扎尕那住宿 | 2-3 | 优先可取消房，确认真实房间和停车 |
| 阿坝县住宿 | 2-3 | 优先热水、供暖、停车、供氧和能帮问景区 |
| 包车/租车 | 3 | 比车型、费用包含项、夜路态度、保险和取消 |
| 核心门票 | 2 个来源以上 | 官方/OTA/酒店或景区电话交叉确认 |

候选进入 `shortlist` 前，必须满足：

- [ ] 有最终含税价或总价。
- [ ] 有退改/取消规则。
- [ ] 有截图或文字证据。
- [ ] 符合摄影行程早出晚归、不赶夜路、停车、热水、供暖、开放确认的要求。
- [ ] 有下一次复核时间或下一步动作。
