## 先看结论

| 场景 | 结论 | 是否改行程 |
|---|---|---|
| 扎尕那晨雾石城 | 全程最值得早起的日出窗口 | 必保 D4 日出 |
| 黄河九曲第一湾日落 | 全程最值得卡时间的日落窗口 | D5 必须 17:00 前到唐克区域 |
| 莲宝叶则湖泊石山蓝调 | 值得等，但只在闭园和身体允许时执行 | 不为蓝调违反闭园 |
| 拉卜楞寺转经长廊 | 下午侧光和清晨都可拍，人物节奏比太阳角度更重要 | 不单独为天象改 |
| 甘加白石崖 | 云影、侧光、牛羊尺度比精确日出日落更重要 | 不熬夜不赶夜路 |
| 星空/银河 | 高原天黑条件不错，但安全、闭园、高反和夜路成本高 | 不作为刚需 |

## 离线生成光线表

出发日期确定后，在仓库根目录运行：

```bash
python3 scripts/gannan_light_windows.py --start 2026-09-12 --days 9
```

输出 CSV：

```bash
python3 scripts/gannan_light_windows.py --start 2026-09-12 --days 9 --format csv
```

脚本会按 9 天路线依次使用兰州、夏河、合作、扎尕那、唐克、阿坝县、莲宝叶则等点位，给出晨蓝调、日出、早黄金、晚黄金、日落和晚蓝调。它是规划估算，不替代当天手机天气 App、摄影 App、山体遮挡和景区闭园时间。

## 星空和银河判断

甘南和阿坝高原暗空条件比城市好，但本线路不建议把星空作为刚需：

- 扎尕那和莲宝叶则夜间机位涉及山路、村道、湖边、闭园和高反风险。
- 6-8 月午后到夜间对流天气多，云量不稳定。
- 月相、云量、透明度、风和安全停车点必须同时成立才值得执行。
- 如果要拍，只在酒店附近、安全空地、司机和酒店确认可行的地方短时间试拍。

## 月相和星空复核

出发前 T-7、T-3、T-1 复核：

- [ ] 月相和月落时间：月亮太亮则不追银河。
- [ ] 云量和透明度：只看天气晴不够，要看夜间云量。
- [ ] 风力和体感温度：高原夜间失温风险高。
- [ ] 安全停车点：不能停在路中、弯道、村道窄处。
- [ ] 是否闭园/禁入/禁飞：景区规则优先。

## 现场 Star Walk / Planit 设置

```text
1. 先设置当前位置，不要只用兰州。
2. 日期切到实际拍摄夜晚。
3. 看银河核心方向是否与安全前景同向。
4. 看月亮是否在天上、月落时间是否足够早。
5. 如果需要夜间离开酒店，先问司机和酒店，不临时单独行动。
```

## 蓝调执行边界

| 地点 | 可等蓝调吗 | 边界 |
|---|---|---|
| 米拉日巴佛阁 | 可以 | 广场安全，拍完回合作酒店 |
| 扎尕那 | 可以短等 | 只在已确认安全机位，不夜间换点 |
| 黄河九曲 | 谨慎 | 看闭园和下撤拥堵，不把最后一班交通卡死 |
| 莲宝叶则 | 谨慎 | 景区闭园、身体状态和返程安全高于蓝调 |
