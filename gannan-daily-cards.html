这份是手机现场版。每天只看：今日目标、硬截止、主机位、砍点顺序和睡前确认。详细机位参数看 甘南莲宝叶则_摄影手册.md。 每天要带回哪些成片素材，看 甘南莲宝叶则_大片交付清单.md。

全程三条铁律

1. 扎尕那全天和莲宝叶则全天不能压缩。
2. 午后雷雨、路滑、高反、景区临时调整时，先砍顺路小点，不赶夜路。
3. 每晚确认第二天：天气、门票/开放、出发时间、油量、早餐、砍点顺序。

D0｜兰州抵达

项目 执行
今日目标 到兰州、取车/见司机、补给、睡好。
不做 不夜游、不熬夜、不喝酒。
必查 车辆、保险、离线地图、天气、酒店、D1 出发时间。
睡前 相机充电、卡格式化、证件/订单截图离线保存。

D1｜兰州 → 夏河｜拉卜楞寺 + 桑科草原

项目 执行
今日目标 进入甘南，适应海拔，拍拉卜楞和草原日落。
建议出发 08:00 兰州出发。
主机位 拉卜楞寺转经长廊、贡唐宝塔/寺院外侧高点、桑科草原日落。
硬截止 20:30 前回夏河休息。
砍点顺序 先砍桑科深度停留，再砍贡唐宝塔；拉卜楞外观和转经长廊优先。
睡前 确认 D2 甘加道路、合作酒店、早餐/早出门。

D2｜夏河 → 甘加秘境 → 合作

项目 执行
今日目标 拍甘加地貌，晚上住合作，为扎尕那留体力。
建议出发 08:30 夏河出发；若补拉卜楞清晨，控制到 08:00 前结束。
主机位 八角城、白石崖、米拉日巴佛阁蓝调。
硬截止 甘加不要拖到天黑；15:00-17:00 到合作休整。
砍点顺序 先砍八角城深度，再砍米拉日巴入内；白石崖和合作住宿优先。
睡前 确认扎尕那预约/门票、民宿定位、停车、晚餐、D3 日落点。

D3｜合作 → 扎尕那｜抵达石城日落

项目 执行
今日目标 天黑前到扎尕那，只追一个日落机位。
建议出发 08:00 合作出发。
主机位 达日观景台或业日村附近，二选一。
硬截止 抵达后不来回跑；天黑后不换机位。
砍点顺序 路上小停靠全砍；扎尕那日落和入住优先。
睡前 确认 D4 日出点、出门时间、早餐、栈道/仙女滩开放。

D4｜扎尕那全天

项目 执行
今日目标 本行程甘南段核心日：晨雾、石城、村寨、栈道。
建议出发 日出前 30-45 分钟到位。
主机位 达日/业日/东哇晨雾，仙女滩/栈道，村寨傍晚。
硬截止 中午强光休息；雨后不进危险沟谷。
砍点顺序 先砍峡谷深度，再砍仙女滩；日出和傍晚石城优先。
睡前 确认 D5 08:00 前离开、郎木寺轻拍、花湖/黄河九曲门票和唐克住宿。

D5｜扎尕那 → 郎木寺 → 花湖 → 黄河九曲

项目 执行
今日目标 上午轻拍郎木寺/花湖，傍晚保黄河九曲日落。
建议出发 08:00 前离开扎尕那，除非日出补拍极好且路况允许。
主机位 郎木寺外观、若尔盖花湖、黄河九曲第一湾日落。
硬截止 17:00 前到唐克/黄河九曲区域。
砍点顺序 先砍郎木寺深度，再砍花湖停留；黄河日落优先。
睡前 确认 D6 到阿坝县路线、油量、阿坝县酒店、莲宝叶则开放初查。

D6｜唐克 / 若尔盖 → 阿坝县

项目 执行
今日目标 安全到阿坝县，高海拔缓冲，给莲宝叶则留体力。
建议出发 08:30 出发；前晚黄河失败才考虑短时间补晨光。
主机位 顺路草原、公路、经幡、月亮湾/瓦切塔林可选。
硬截止 下午到阿坝县，不夜路。
砍点顺序 所有顺路景点都可砍；阿坝县休整和补给优先。
睡前 确认莲宝叶则门票、开放范围、最早入园、最晚离园、天气、早餐打包。

D7｜莲宝叶则全天

项目 执行
今日目标 全程第二核心日：湖泊倒影、石山压缩、高位观景台、蓝调。
建议出发 06:30 阿坝县出发，按季节日出和景区开放调整。
主机位 扎尕尔措、湖边石山、高位观景台/4520 方向、蓝调。
硬截止 服从景区闭园；高反、雷雨、风大立刻降强度。
砍点顺序 先砍高位观景台，再砍蓝调；湖泊石山安全图优先。
睡前 备份照片，确认 D8 返程分段终点、路况、油量和住宿。

D8｜阿坝县 → 合作 / 临夏

项目 执行
今日目标 长返程第一段，只顺路记录，不加新景区。
建议出发 08:00-08:30。
主机位 车窗、草原、公路、经幡，安全停车才拍。
硬截止 不从阿坝县直冲兰州机场，不疲劳驾驶。
砍点顺序 所有拍摄都可砍；白天安全抵达合作/临夏优先。
睡前 确认 D9 回兰州时间、还车、航班/高铁、行李和存储卡。

D9｜合作 / 临夏 → 兰州

项目 执行
今日目标 还车、返程，不加刚性景点。
建议出发 按航班/高铁倒推，保守到兰州。
主机位 无；只拍收尾纪实。
硬截止 返程前 3 小时到机场或高铁站更稳。
必查 车身、油量、后备箱、座椅下、相机电池、存储卡、硬盘。

每晚 5 分钟复盘

1. 明天最重要的一张照片是什么？
2. 明天第一个要砍的点是什么？
3. 明天几点出发，几点必须到终点？
4. 门票/开放/观光车是否确认？
5. 照片是否双备份，电池是否全充？
