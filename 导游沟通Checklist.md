# 每日导游/司机沟通 Checklist（中英文话术）

独立用途：这份文档专门放“每天出发前和现场要跟司机、向导、船家确认什么”。中文用于自己核对，英文可直接发给对方。网页版本可直接打勾，勾选状态保存在当前浏览器本地。

总原则：

1. 安全优先；天气、路况、火山限制、海况不合适时，按司机/向导/船家判断改备选。
2. 每天出发前确认：接人时间、目的地 pin、返回硬截止、票/船/向导、天气/海况风险。
3. 用 `Could you please...`、`If it is safe and allowed...`、`Please let us know...` 这类表达，不用命令式语气。
4. Bromo、Ijen、Penida 都不要把高风险项目说成“必须完成”；要说“如果安全且允许”。

## 通用礼貌话术

| 场景 | 中文 | English |
|---|---|---|
| 安全优先 | 今天安全第一。如果天气、路况或规定不适合，我们可以改备选。 | Safety comes first today. If the weather, road conditions, or local rules are not suitable, we can switch to the backup plan. |
| 请发定位 | 麻烦发今天第一个目的地的准确 Google Maps pin。 | Could you please send the exact Google Maps pin for our first destination today? |
| 请提前提醒 | 如果某个点不安全、关闭或太赶，请提前告诉我们。 | Please let us know early if any place is unsafe, closed, or too rushed. |
| 摄影需求 | 我们会拍风景照，可能需要提前到机位和使用三脚架。 | We will shoot landscape photos, so we may need to arrive early and use tripods. |
| 不懂时用翻译 | 如果英文不清楚，我们可以用 WhatsApp 或 Google Translate 确认。 | If English is not clear, we can confirm by WhatsApp text or Google Translate. |
| 硬截止 | 今天有硬截止时间，请帮我们按时返回或出发。 | We have a hard deadline today. Please help us leave or return on time. |
