这份清单不再解释“怎么拍”，只回答“回来必须带回哪些片”。现场每天先保封面级照片，再补故事图、人物尺度、短视频和安全图。天气失败时，不空手回酒店，要按替代清单带回能成片的素材。

## 最终交付目标

| 类型 | 数量 | 标准 |
|---|---:|---|
| 封面级主图 | 12-18 张 | 可以单独发朋友圈/小红书/相册封面，构图完整，光线或情绪明确 |
| 每日故事图 | 45-70 张 | 能串起兰州出发、进入甘南、高原转场、扎尕那、黄河、莲宝叶则和返程 |
| 人物尺度图 | 12-20 张 | 人、车、经幡、栈道或村寨作为尺度，不抢主体 |
| 短视频镜头 | 30-50 条 | 每条 5-12 秒，稳定、干净、有起承转合 |
| 失败天气备选图 | 每个核心日 3-5 张 | 雨雾、白墙天、风大、无倒影时仍能表达现场 |
| 订单/现场证据截图 | 全程留存 | 门票、酒店、车辆、天气、开放、航班/高铁和聊天记录 |

## 封面级主图池

优先把下面这些拍到。若天气不配合，至少每个 S/A 级地点带回一张“安全图”。

| 优先级 | 画面 | 地点 | 成片标准 | 失败替代 |
|---|---|---|---|---|
| S | 扎尕那晨雾石城 | 达日/业日/东哇 | 石峰、村寨、晨雾、光线至少有三项成立 | 雨雾村寨、屋顶、牛羊和道路层次 |
| S | 莲宝叶则石山湖泊 | 扎尕尔措/高位观景台 | 石山轮廓清楚，湖泊或云影有层次 | 长焦压石山、湖岸线、水面纹理 |
| A | 拉卜楞转经长廊 | 拉卜楞寺外侧 | 人物节奏、转经筒、侧光和背景干净 | 手部细节、背影、长廊透视 |
| A | 黄河九曲日落 | 唐克高位 | 河面有金色或蓝调层次，弯道清楚 | 栈道剪影、长焦河曲、云层压暗画面 |
| A | 白石崖草原 | 甘加秘境 | 白石崖、草原、牛羊/人车尺度同时成立 | 极简草线、公路、白崖纹理 |

## 每日交付清单

### D0 兰州抵达

- [ ] 取车/见司机记录图：车、行李、装备。
- [ ] 补给图：水、氧气、药品、雨具、保暖。
- [ ] 订单证据：车辆、酒店、D1 天气和路线截图。
- [ ] 收纳图：相机、镜头、卡、硬盘、电池整理完成。

### D1 拉卜楞寺 + 桑科草原

- [ ] 封面候选：拉卜楞转经长廊人物节奏。
- [ ] 封面候选：贡唐宝塔/寺院屋顶长焦压缩。
- [ ] 封面候选：桑科草原晚光、马/牛羊/人像尺度。
- [ ] 故事图：兰州出发、进甘南路牌、夏河街道、寺院外观。
- [ ] 细节图：转经筒、手部、墙面纹理、经幡、藏袍背影。
- [ ] 视频：转经长廊横移、草原云影延时、车辆驶过公路。
- [ ] 失败替代：雨天拍寺院屋檐、水痕、转经手部特写。

### D2 甘加秘境 + 合作蓝调

- [ ] 封面候选：白石崖 + 牛羊/车辆尺度。
- [ ] 封面候选：八角城几何线条。
- [ ] 封面候选：米拉日巴佛阁蓝调。
- [ ] 故事图：夏河清晨、甘加路、合作市区补给。
- [ ] 细节图：草原道路、崖壁纹理、经幡、停车点环境。
- [ ] 视频：白石崖云影、车窗路景、佛阁蓝调静镜。
- [ ] 失败替代：阴天拍白石崖极简黑白感、草线和公路。

### D3 合作到扎尕那日落

- [ ] 封面候选：扎尕那石城第一眼。
- [ ] 封面候选：村寨、石峰、炊烟或云影长焦层次。
- [ ] 故事图：长转场、进山路、抵达民宿、停车和晚餐。
- [ ] 人物尺度：人在观景点边缘，不挡石峰主体。
- [ ] 视频：抵达扎尕那的第一段推镜/摇镜。
- [ ] 失败替代：浓雾时拍路灯、屋顶、村路、雨雾中的人影。
- [ ] 安全图：即使没有好光，也拍一张清晰的扎尕那全景。

### D4 扎尕那全天

- [ ] S 级主图：晨雾石城。
- [ ] S 级主图：村寨 + 石峰 + 光束或云缝。
- [ ] A 级主图：仙女滩/栈道人物环境。
- [ ] 故事图：清晨出门、头灯、早餐、村寨生活、傍晚回民宿。
- [ ] 细节图：屋顶、炊烟、牛羊、草坡、栈道曲线。
- [ ] 视频：晨雾流动、村寨静镜、栈道行走、云影扫山。
- [ ] 失败替代：白墙雾拍近景、背影、雨伞、木屋、牛羊。
- [ ] 安全图：日出、上午、傍晚各保一张扎尕那环境图。

### D5 郎木寺 + 花湖 + 黄河九曲

- [ ] B 级主图：郎木寺/红石崖/纳摩峡谷轻拍。
- [ ] B 级主图：花湖湿地云影或鸟。
- [ ] A 级主图：黄河九曲第一湾日落。
- [ ] 故事图：扎尕那离开、郎木寺镇、若尔盖草原、唐克住宿。
- [ ] 人物尺度：栈道剪影、观景台边缘、远处人群。
- [ ] 视频：草原车窗、花湖风声、黄河日落延时。
- [ ] 失败替代：黄河无日落时拍蓝调河曲和栈道剪影。
- [ ] 安全图：17:00 前到不了唐克时，拍顺路草原作为当天主故事。

### D6 唐克 / 若尔盖到阿坝县

- [ ] 故事图：高原缓冲日、草原公路、经幡、黄河支流。
- [ ] 人物尺度：车、人、经幡和远山关系。
- [ ] 补给图：加油、买水、药品、阿坝县酒店。
- [ ] 视频：公路前进、经幡风声、阿坝县街道。
- [ ] 证据图：莲宝叶则开放确认、天气截图、门票/预约截图。
- [ ] 失败替代：这天本来就是缓冲日，不追硬片，保证睡眠。

### D7 莲宝叶则全天

- [ ] S 级主图：扎尕尔措/湖泊倒影。
- [ ] S 级主图：石山湖泊长焦压缩。
- [ ] A 级主图：高位观景台/4520 方向云影。
- [ ] A 级主图：蓝调石山湖泊，前提是闭园和身体允许。
- [ ] 故事图：阿坝县清晨出发、景区入口、观光车/道路、补给和撤离。
- [ ] 人物尺度：人在湖岸安全区域或观景台边，不越界。
- [ ] 视频：湖面风、石山摇镜、脚步声、蓝调静镜。
- [ ] 失败替代：风大无倒影时拍湖岸线、水面纹理、石山细节。
- [ ] 安全图：刚进景区先拍一张清晰可用的石山湖泊环境图。

### D8 阿坝县到合作 / 临夏

- [ ] 故事图：长返程、公路、经幡、车窗、午餐、住宿。
- [ ] 人物尺度：路边安全停车的人与草原。
- [ ] 视频：返程车窗、路牌、云影。
- [ ] 证据图：D9 返程交通、还车/机场/高铁安排截图。
- [ ] 失败替代：所有拍摄都可砍，安全到中转住宿就是当天目标。

### D9 回兰州

- [ ] 收尾图：兰州牛肉面、车站/机场、行李、硬盘和存储卡。
- [ ] 证据图：还车车身、油量、后备箱、订单结清。
- [ ] 视频：机场/高铁站收尾、行李推进、最后一张路线回看。
- [ ] 数据图：照片双备份完成，卡和硬盘都在身边。

## 短视频镜头清单

| 类型 | 画面 | 备注 |
|---|---|---|
| 开场 | 兰州取车、行李、地图、出发路牌 | 3-5 个短镜头即可 |
| 路线 | 车窗、隧道、草原公路、经幡 | 每天至少 2 条 |
| 人文 | 转经、背影、手部、街道、早餐 | 尊重拍摄边界 |
| 风光 | 晨雾、云影、河曲、湖面、石山 | 每个核心机位保 2 条 |
| 声音 | 风声、脚步、转经筒、车门、雨声 | 现场声比配乐更有记忆点 |
| 收尾 | 还车、机场/高铁、硬盘备份 | 给游记结尾用 |

## 后期选片规则

| 星级 | 标准 | 动作 |
|---|---|---|
| 5 星 | 可以做封面，光线、主体、构图、情绪都成立 | 进入主图池，优先精修 |
| 4 星 | 构图和信息完整，但光线或情绪略弱 | 做组图或游记 |
| 3 星 | 记录价值强，画面普通 | 做故事图 |
| 2 星 | 重复、构图乱、信息弱 | 只保留必要记录 |
| 1 星 | 糊、过曝、危险位置、侵犯边界 | 删除或不发布 |
