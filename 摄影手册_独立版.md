# 印尼摄影手册独立版

更新日期：2026-06-10
适用行程：2026-06-13 至 2026-06-20
对应主行程：[行程_完整增强版.md](行程_完整增强版.md)
时区口径：东爪哇用 WIB, UTC+7；巴厘岛和 Nusa Penida 用 WITA, UTC+8。

这份文件是从主行程独立出来的摄影执行手册。主行程管交通、住宿和大节奏；本手册只管现场怎么拍、带什么、查什么、什么时候放弃。

## 先看结论

1. 天气和云图是强时效信息。出发前 24-48 小时和当天清晨必须再查 BMKG。
2. 日出日落是稳定信息，按天文时间排计划；山区遮挡会让实际见光晚 5-20 分钟。
3. Bromo 主片用 70-200mm，不是超广角。14-24mm 留给星空、沙海强前景和瀑布谷底。
4. ND 的刚需在 D2 瀑布和 Nusa Penida 海浪慢门，不在 Bromo / Ijen。Bromo 和 Ijen 优先三脚架、保暖、防尘、防硫磺。
5. Ijen 蓝火只作为条件项目。若官方/向导不允许下坑、风把硫磺吹向人、体力不够，立刻转为 rim 日出和矿工人文。
6. Nusa Penida 不要为照片下危险台阶或靠无护栏崖边。东线的胜负在早出发和高位机位，不在强行下海滩。

## 现场放弃规则

| 情况 | 立刻怎么做 |
|---|---|
| 瀑布下雨、水量大、向导摇头 | 不下谷底，只拍上方，提前转场 |
| Bromo 低云白墙 | 不乱换机位，原地等短开窗；拍人群和雾中山脊补故事图 |
| Bromo 官方 1 km 限制未解除 | 不去火山口阶梯/口沿，最多 Pura Luhur Poten / 沙海外围 |
| Ijen 硫磺风吹向人 | 戴紧面具，停止拍摄，上撤到上风侧 |
| Ijen 蓝火不开放 | 改拍 rim 蓝调、湖、矿工，别把失败感带到后半程 |
| Nusa Penida 外海涌浪大 | 浮潜改点或取消，主相机不下船裸拍 |
| Angel's Billabong 非低潮或浪涌 | 不靠近潮池，不下水 |
| 崖边风大/无护栏 | 不后退取景，不用 14mm 贴边冒险 |

1. 外袋：头灯、镜头布、雨罩、CPL/ND、手套、纸巾、充电宝。
