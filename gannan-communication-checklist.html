甘南 + 莲宝叶则沟通 Checklist

甘南 + 莲宝叶则司机 / 酒店 / 景区沟通 Checklist

对应主攻略：甘南莲宝叶则_完整攻略.md
对应核验：甘南莲宝叶则_核验记录.md

这份文件只放现场能直接复制使用的话术和确认项。重点是让司机、酒店、民宿和景区工作人员明白：这是摄影行程，不是打卡行程；可以砍点，但不赶夜路、不违规、不冒险。

总话术

给司机 / 包车师傅

师傅您好，我们这趟主要是摄影，不是普通打卡。
每天最重要的是日出、日落、扎尕那和莲宝叶则的核心机位。
如果天气、路况、景区政策不合适，我们可以砍小景点，但不要赶夜路，也不要走危险路。
请您每天晚上帮我们一起确认第二天出发时间、路况、加油、停车点和景区是否开放。

给酒店 / 民宿

您好，我们是摄影行程，可能清晨 5 点左右出门。
请问停车是否方便？早上能否提前出门？热水和供暖是否稳定？
如果方便，也请帮忙问一下明天景区是否正常开放、是否需要预约、观光车最早几点。

给景区 / 售票处

您好，请问今天景区是否全线开放？
门票是否需要实名预约？是否必须乘坐观光车？
最早入园和最晚离园时间是几点？
哪些区域不能拍照、不能飞无人机、不能进入？

D0 兰州

取车 / 包车确认

话术

明天我们第一站去夏河拉卜楞寺，不赶扎尕那。
请您帮忙按 08:00 左右出发安排，路上如果有安全的观景点可以短暂停车，但不要影响下午拍寺院和草原日落。

D1 夏河 / 拉卜楞寺

酒店确认

寺院拍摄边界

话术

我们主要拍寺院外观、转经长廊和贡唐宝塔方向。
寺院内部、佛像和不允许拍摄的地方我们不拍。
如果有法会或管理要求，请直接提醒我们。

D2 甘加秘境 / 合作

给司机

话术

今天甘加秘境我们不深度徒步，主要拍八角城、白石崖和草原路。
下午要到合作休整，晚上拍米拉日巴佛阁蓝调，所以甘加不要拖到太晚。
如果天气不好，我们可以缩短甘加，保证合作住宿和明天去扎尕那。

D3 合作到扎尕那

出发前确认

给民宿

您好，我们今天下午到扎尕那，想拍日落和明早日出。
请问停车在哪里？到达后去达日观景台/业日村拍照是否方便？
明天清晨 5 点左右能否出门？早餐能否延后或打包？

给司机

今天到扎尕那后，我们只拍一个日落机位，不来回跑。
请您根据停车、路况和天气帮我们选达日观景台或业日村附近。
天黑以后不再换点。

D4 扎尕那全天

清晨确认

话术

今天扎尕那是重点，我们优先拍日出晨雾和傍晚石城。
中午强光时可以休息，不需要硬塞很多点。
如果下雨或栈道湿滑，我们就拍村寨和近景，不进危险沟谷。

村寨拍摄

我们只在公共道路和观景点拍摄。
如果要拍人、院落或近距离生活场景，会先征得同意。
不方便拍的地方请直接告诉我们。

D5 郎木寺 / 花湖 / 黄河九曲

关键时间

给司机

今天黄河九曲第一湾日落最重要。
郎木寺和花湖都是顺路拍摄，不能影响 17:00 前到唐克。
如果时间不够，请优先砍花湖停留时间，保证黄河日落。

给景区

请问花湖今天观光车和栈道是否正常开放？
黄河九曲第一湾最晚入园和最晚下撤是几点？
门票是否包含扶梯/观光车，还是可以单独选择？

D6 唐克 / 若尔盖到阿坝县

目标

给酒店

您好，我们明天一早去莲宝叶则。
请问从酒店到景区大概多久？最近景区是否正常开放？
能否帮忙确认明天门票、观光车/自驾进入规则、最早入园和最晚离园时间？
早上能否提前出门，早餐是否可以打包？

D7 莲宝叶则

出发前确认

给景区

您好，请问今天莲宝叶则哪些区域开放？
扎尕尔措、高位观景台、4520 方向是否都可以去？
景区内交通怎么安排？最晚几点必须离开？
湖边、观景台和无人机有哪些限制？

给司机

今天莲宝叶则是重点，我们希望尽量早进景区，慢慢拍湖泊、石山和蓝调。
如果天气变差、景区要求离开或有人高反，我们马上降低强度或返回。
请不要为了照片走不允许的路，也不要超过景区闭园时间。

高反应对

如果有人头痛、胸闷、恶心、走路不稳，我们立刻停止拍摄，回车休息或下降海拔。
氧气只是临时缓解，不用来硬撑继续爬高。

D8 阿坝县到合作 / 临夏

给司机

今天是长返程第一段，只拍顺路点，不加新景区。
请优先安全和白天行车，如果天气、路况或体力不好，我们就早点到合作/临夏休息。

确认

D9 回兰州

还车 / 返程

话术

今天只回兰州，不加刚性景点。
请按航班/高铁时间保守安排，宁可早到，也不要卡点。

每晚复盘话术

我们确认一下明天三件事：
1. 明天最重要的一个拍摄目标是什么？
2. 如果天气或路况不好，先砍哪个点？
3. 明天几点出发、几点必须到酒店/景区/机场？
