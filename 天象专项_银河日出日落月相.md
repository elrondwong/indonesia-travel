# 印尼行程天象专项：银河、日出、日落、月相

更新日期：2026-06-10  
适用行程：2026-06-13 至 2026-06-20  
时区口径：东爪哇为 WIB, UTC+7；巴厘岛和 Nusa Penida 为 WITA, UTC+8。

这份文档只回答一个问题：本行程里除了常规日出日落，还有哪些点值得为了银河、月牙、日月对位额外安排时间。

## 先看结论

| 优先级 | 场景 | 结论 | 是否改行程 |
|---|---|---|---|

| A | Nusa Penida 东线日出 | 6/19 Diamond / Thousand Islands / Rumah Pohon 很值得，日出 06:30 WITA，方位 66 度。 | 保持 05:30-05:45 出发 |
| B | Nusa Penida 西线 Kelingking 日落 + 月牙 | 6/18 Kelingking 可拍西向金色光和日落，日落 18:07 WITA，月牙在西北高空；Paluang 是同组侧面候选。 | 让旅行社保 17:30-18:10 Kelingking 窗口 |
| B | Bromo 第二晚银河 | 6/15 23:30-6/16 00:05 也可拍，但会影响 D4 转场和 D5 Ijen 体力。 | 不建议连续熬夜 |

| C | Ijen 银河 | 天文条件可以，但方向、安全、硫磺烟和体力都不划算。 | 不为银河改 Ijen |
| C | Kapas Biru / Tumpak Sewu 顶日顶月 | Semeru 在北方，太阳和月亮主要在东西方向，不构成顶日顶月。 | 不改 |

最重要的现实判断：**本行程没有一个“太阳或月亮精确压在火山/海蚀岩尖顶”的强对位。** 真正强的是 Bromo 银河对位、Penida 东线日出、Penida 西线日落/月牙。

## 方法和核验口径

1. 先用地图点位确定相机位置和目标位置，再计算真北方位角。
2. 太阳、月亮和银河核心用 Skyfield + JPL DE421 星历本地计算；日出日落和月相用 Timeanddate 交叉核验。
3. 天气和云图用 BMKG。BMKG 乡镇预报适合看趋势，Himawari 云图适合出发前 0-6 小时临场判断。
4. 所有方位角都是真北方位角。手机指南针现场会被车、三脚架、火山磁性岩和金属栏杆影响，只作辅助。

## Day 2｜6/14｜Kapas Biru / Tumpak Sewu

| 项目 | 结论 |
|---|---|
| 值得拍银河吗 | 不值得。D2 晚上你已经需要转到 Bromo，银河窗口应该留给 Bromo。 |
| 值得拍顶日吗 | 不值得。Semeru 从 Kapas 看约 350 度，从 Tumpak 看约 2 度；6/14 日出方位约 67 度。 |
| 值得拍顶月吗 | 不值得。6/14 是新月前后，月亮极细且白天靠近太阳。 |
| 保留拍法 | 按原计划拍清晨雾层、瀑布、Semeru 云开。 |
| 天气风险 | BMKG Pronojiwo 对 6/14 附近多为 `Udara Kabur` / 高湿，瀑布水雾重，银河透明度不应期待。 |

执行建议：不要为了天象改 D2。Kapas 和 Tumpak 的胜负在水雾、清晨层次和 Semeru 是否露头，不在日月对位。

### 执行选择

| 方案 | 时间 | 成片预期 | 代价 |
|---|---|---|---|
| 认真银河 | 6/14 22:45-23:00 酒店接，23:20 到 King Kong，拍到 00:15 | 14mm 竖构图或拼接，银河核心高悬在 Batok/Bromo/Semeru 上方 | 牺牲睡眠，日出前会很累 |
| 原计划日出 | 03:15 Jeep 接，04:00 到位 | 蓝调、云海、长焦三峰，银河只作尾声 | 不拍核心对位 |
| 不建议 | 只为 18:00 横银河上山 | 18:00 天还没黑透，核心低，且和火山主方向不对 | 成本高，收益低 |

### Bromo 天气核验

BMKG Tosari / Wonokitri 当前显示 6/14-6/16 一带多为 `Udara Kabur`，Wonokitri 温度约 14-21 C、湿度常到 90% 上下。这不等于一定没星，但意味着银河对比度风险高。出发前必须看 BMKG Himawari：

- 黑色/蓝色：云发展少，可以执行银河。
- 橙色/红色：高冷云顶和对流云明显，取消银河，保日出。
- 低云白墙但星星能见：可以先拍星空，再等日出开窗。

## Day 4｜6/16｜Bromo 补拍

6/15 晚到 6/16 凌晨的银河对位仍然存在：23:30 核心约 175 度，23:45 约 184 度，00:00 约 193 度。天文条件并不差，月亮也几乎无干扰。

但不建议把 D4 再做成银河主拍，因为 D4 要长转场到 Banyuwangi，D5 又是 Ijen 夜爬。除非 D3 天气完全失败、你愿意牺牲睡眠，否则 D4 应该只保早晨补拍。

## Day 5｜6/17｜Ijen

| 项目 | 结论 |
|---|---|
| 日出 | 值得，已在主计划里。Ijen 日出约 05:34 WIB，方位约 67 度。 |
| 银河 | 不建议。03:30 银河核心约 241 度，高度约 30 度，方向偏西南；你主要在夜爬、硫磺烟、火山湖安全判断中。 |
| 月亮 | 6/17 月牙约 6%-8%，白天升起、晚上西落，不帮 Ijen 主画面。 |
| 风险 | 硫磺烟、护目、防毒面具、向导开放范围比天象更重要。 |

执行建议：Ijen 不为银河改计划。蓝火开放就拍蓝火；蓝火不开放就拍 rim 蓝调、火山湖和矿工。

## Day 6｜6/18｜Nusa Penida Kelingking 日落和月牙

| 项目 | 数据 |
|---|---|
| 日落 | 18:07 WITA，方位约 294 度 |
| 天文暮光结束 | 19:22 WITA |
| 月亮 | 约 16% 盈凸前的新月牙，21:27 WITA 落下，落点约 288 度 |
| 18:30 月亮 | 方位约 303 度，高度约 39 度 |
| 19:00 月亮 | 方位约 299 度，高度约 33 度 |

### 选点

| 点位 | 适合度 | 取舍 |
|---|---|---|
| Kelingking Viewpoint / Paluang Cliff | A | 旅行社最容易配合，西向崖线适合日落和金色光；Paluang 是同组侧面机位，可下午先拍白天景色并踩日落点；Kelingking 可下上段安全台阶拍沙滩视角。 |
| Crystal Bay | B | 如果 Kelingking 因拥堵/天气取消，Crystal Bay 是最稳替代。 |
| Banah Cliff | B- | 画面更野，但道路、天黑返程和崖边安全压力更大，只作有门槛快速支线，不抢回 Kelingking 日落。 |

执行建议：上午先浮潜，14:00 前结束并切回陆地段；下午按 Angel's Billabong → Broken Beach → Kelingking Beach → Paluang Cliff 白天景色/日落踩点 → Banah Cliff Point → 回 Kelingking Beach 日落执行。Banah Cliff Point 只作路况好时的快速支线，17:30 前回不到 Kelingking 就取消。Kelingking 可以下到上段安全台阶拍沙滩视角；完整下到底部沙滩会取消 Paluang / Banah / 回拍日落。不要为了月牙在无护栏海崖逗留到很晚。

## Day 6 夜｜6/18-6/19｜Nusa Penida 银河

| 时间 WITA | 银河核心方位/高度 | 判断 |
|---|---|---|
| 6/18 20:00 | 118 度 / 27 度 | 东南，横银河可拍 |
| 6/18 23:30 | 153 度 / 67 度 | 转向南偏东，高度很高 |
| 6/19 00:00 | 170 度 / 69 度 | 接近正南，竖构图很强 |
| 6/19 00:30 | 189 度 / 70 度 | 正南到南偏西 |
| 6/19 03:30 | 239 度 / 40 度 | 已偏西南 |

月亮 21:27 后落下，银河条件很好。但现实问题是：Nusa Penida 夜间路窄、坑多、海崖无护栏，很多漂亮点不适合夜里去。  

执行建议：只在 **Coco Resort 附近安全空地** 或司机确认的安全开阔点试拍 15-30 分钟。不要夜里去 Banah、Kelingking、Tembeling、Broken Beach 追银河。

## Day 7｜6/19｜Nusa Penida 东线日出

| 项目 | 数据 |
|---|---|
| 日出 | 06:30 WITA，方位约 66 度 |
| 天文晨光开始 | 05:15 WITA |
| 月亮 | 约 22%-25%，白天升起，晚上落下，不影响日出 |
| BMKG Pejukutan | 当前 6/18-6/19 附近多为 `Cerah` / `Cerah Berawan`，比东爪哇山地更友好 |

### 选点

| 点位 | 适合度 | 取舍 |
|---|---|---|
| Thousand Islands / Rumah Pohon | A | 东向海岛群，日出方向在左前方，最适合日出故事图。 |
| Diamond Beach 高位 | A- | 白崖、台阶、海水颜色强，日出氛围好；太阳不一定压在海蚀岩上。 |
| Atuh Beach 高位 | B+ | 同属东线日出点，适合作为 Diamond 后的第二点。 |

执行建议：05:30-05:45 从 Coco Resort 出发太晚只够赶 06:30 左右的第一光。如果特别想卡日出前蓝调，和司机商量 **05:10-05:20 出发**；但不要牺牲安全驾驶。

## Day 7 晚 / Day 8｜Kuta

Kuta 的强项是西向海滩日落，不是银河。6/19 日落约 18:09 WITA，月亮约 25%，18:30 仍在西北高空 52 度左右。  

但你 D7 16:30 从 Nusa Penida 回 Sanur，正常到 Kuta 已经接近日落后，不应把 Kuta 日落作为刚性目标。D8 是返程日，也不建议新增景点。
