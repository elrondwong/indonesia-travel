## 先看结论

1. 甘南的大片不是靠景点数量堆出来的，而是靠云影、晨雾、寺院秩序、草原尺度和长焦压缩。
2. 莲宝叶则的胜负在风小倒影、石山轮廓、蓝调和高海拔体力管理。不要第一二天就冲。
3. 三支镜头够用：14-24mm 拍大空间，24-105mm 做挂机，70-200mm 是甘南和莲宝叶则的主力。
4. 寺院和村寨先尊重，再拍照。能拍外观、背影、手部、环境，不把镜头怼到信众和僧人脸上。
5. 午后雷阵雨、景区关闭、身体高反、路况变差时，优先砍顺路点，保扎尕那全天和莲宝叶则全天。

## 全程器材分工

| 器材 | 结论 | 主要用途 |
|---|---:|---|
| 14-24mm | 带 | 扎尕那栈道、拉卜楞建筑、莲宝叶则湖泊大景、蓝调、星空备选 |
| 24-105mm | 必带 | 全程挂机，路线记录、寺院外观、草原、村寨、湖泊 |
| 70-200mm | 必带 | 转经长廊、白石崖、扎尕那石峰、花湖、黄河九曲、莲宝叶则石山压缩 |
| 三脚架 | 建议必带 | 蓝调、星空、倒影、黄河九曲日落、低 ISO 风光 |
| CPL | 建议带 | 草原云影、湿地、湖泊反光；拍倒影时不要一把拧到最强 |
| ND64 | 可选 | 云流、水面慢门；本线不是刚需 |
| 雨罩/干袋 | 必带 | 午后雷阵雨、湿地风雨、高原突发天气 |
| 头灯 | 建议带 | 扎尕那日出前、莲宝叶则蓝调后、酒店早出门 |

## Day 1｜拉卜楞寺 + 桑科草原

### 机位 1：拉卜楞寺转经长廊

| 项目 | 执行 |
|---|---|
| 时间 | 下午侧光或次日清晨；强光中午只拍记录。 |
| 拍摄对象 | 转经筒、手部、藏袍、信众背影、长廊透视。 |
| 镜头顺序 | 70-200mm 第一，24-105mm 第二。 |
| 焦段 | 100-200mm 拍手和背影；35-70mm 拍长廊环境。 |
| 参数 | 人物 1/250s 以上；静物 f/4-f/8；室外可 -0.3EV 保高光。 |
| 现场动作 | 先站边缘观察人流方向，再找侧光；不逆行挡转经路线。 |
| 放弃条件 | 法会、管理人员提醒、内部殿堂、信众明显抗拒。 |

### 机位 2：贡唐宝塔 / 寺院外侧高点

| 项目 | 执行 |
|---|---|
| 时间 | 日落前 1 小时到蓝调。 |
| 拍摄对象 | 寺院金顶、白墙、远山、夏河县城层次。 |
| 镜头顺序 | 70-200mm 第一，24-105mm 第二。 |
| 拍法 | 长焦压缩金顶和远山；蓝调时补一张广一点的环境图。 |
| 放弃条件 | 风雨、寺院管理提醒、平台拥挤无法安全架脚架。 |

### 机位 3：桑科草原

| 项目 | 执行 |
|---|---|
| 时间 | 18:00-20:00，按季节日落调整。 |
| 拍摄对象 | 草线、云影、牛羊马、公路弯道、经幡。 |
| 镜头 | 24-105mm 拍环境，70-200mm 压云影和马。 |
| 拍法 | 低机位让草线做前景；人像放在画面 1/3 处，远山不要切头。 |
| 放弃条件 | 雷雨、草地泥泞、经营区不允许进入、马匹不受控。 |

## Day 2｜甘加秘境 + 米拉日巴佛阁

### 机位 1：八角城

| 项目 | 执行 |
|---|---|
| 时间 | 上午到中午，云影比太阳角度更重要。 |
| 拍摄对象 | 城墙几何、村落、草原、远山。 |
| 镜头 | 24-105mm；有合法无人机条件再考虑航拍。 |
| 拍法 | 找高一点的安全路边或观景点，拍城墙线条和远山关系。 |
| 放弃条件 | 现场禁飞、风大、没有明确报备、路边停车危险。 |

### 机位 2：白石崖

| 项目 | 执行 |
|---|---|
| 时间 | 上午云影或下午侧光。 |
| 拍摄对象 | 白色崖壁、草原、牛羊、人车尺度。 |
| 镜头 | 70-200mm 主力，24-105mm 补环境。 |
| 拍法 | 长焦把白石崖压近，把牛羊或人放到崖壁前当尺度。 |
| 放弃条件 | 白墙天、崖壁无层次、停车点不安全。 |

### 机位 3：米拉日巴佛阁蓝调

| 项目 | 执行 |
|---|---|
| 时间 | 日落后 20-40 分钟。 |
| 拍摄对象 | 金色佛阁立面、蓝调天空、广场人物尺度。 |
| 镜头 | 24-105mm，14-24mm 补建筑透视。 |
| 参数 | 三脚架 ISO100-400, f/8；手持就 ISO1600-3200 保 1/60s 以上。 |
| 放弃条件 | 内部不允许拍摄、三脚架挡路、雨大。 |

## Day 4｜扎尕那全天

### 机位 1：晨雾石城

| 项目 | 执行 |
|---|---|

| 拍摄对象 | 石峰、村寨、晨雾、屋顶、炊烟、光束。 |
| 镜头 | 70-200mm 主力；24-105mm 补全景。 |

| 拍法 | 雾太厚时等开窗；雾太薄时用长焦压缩村寨和石峰。 |
| 放弃条件 | 雷雨、能见度极低且无变化、身体高反。 |

### 机位 2：仙女滩 / 栈道

| 项目 | 执行 |
|---|---|
| 时间 | 10:00-12:30 或 16:00 后。 |
| 拍摄对象 | 草坡、石峰、人物环境、栈道曲线。 |
| 镜头 | 14-24mm 拍大空间；24-105mm 拍人像环境。 |
| 拍法 | 人物不要站正中挡山；用草坡、栈道、溪流带视线。 |

### 机位 3：村寨人文

| 项目 | 执行 |
|---|---|

| 拍摄对象 | 屋顶、炊烟、牦牛、村路、人物背影。 |

## Day 7｜莲宝叶则全天

## 天气失败时怎么拍

| 情况 | 改拍 |
|---|---|
| 扎尕那白墙雾 | 村寨屋顶、牛羊、雨雾中的道路、近景人物背影。 |
| 甘加阴天 | 白石崖纹理、草原极简线条、公路和牛羊。 |
| 花湖风大 | 草浪、栈道剪影、湿地远山，不追水面倒影。 |
| 黄河日落被云挡 | 蓝调河曲、栈道人影、长焦压缩河道线条。 |
| 莲宝叶则风大无倒影 | 石山细节、湖岸线、云影、水面纹理。 |
| 午后雷雨 | 车内休息、酒店备份、晚些再出，不在高点和湖边硬等。 |

## 后期方向

| 场景 | 后期重点 |
|---|---|
| 寺院人文 | 控制高光，保留肤色和藏袍颜色，不把饱和度拉爆。 |
| 草原云影 | 局部加对比和去雾，保留草地层次，天空不要过蓝。 |
| 扎尕那晨雾 | 轻微去雾 + 保留雾气，不做成硬 HDR。 |
| 黄河九曲 | 金色水面和暗部河岸分开处理，避免整体橙黄。 |
| 莲宝叶则 | 石山冷暖对比、湖面层次、蓝调色温；保持高原清冷感。 |
