# 甘南 + 莲宝叶则机位核验表

更新日期：2026-06-25  
对应摄影手册：[甘南莲宝叶则_摄影手册.md](甘南莲宝叶则_摄影手册.md)  
对应机位地图：[甘南莲宝叶则_机位地图索引.md](甘南莲宝叶则_机位地图索引.md)  
机位 CSV：[data/gannan_photo_spots.csv](data/gannan_photo_spots.csv)

这份表把“拍大片”的核心机位从攻略文字变成结构化核验清单。每个机位都要有优先级、地图关键词、黄金时间、主力镜头、必带回画面、核验来源、风险和放弃条件。

## 先看结论

检查机位主数据：

```bash
python3 scripts/check_gannan_photo_spots.py
```

脚本会检查：

- S 级机位是否覆盖扎尕那和莲宝叶则。
- A 级以上机位是否覆盖拉卜楞、白石崖和黄河九曲。
- D1/D2/D3/D4/D5/D7 是否都有核心机位。
- 每个机位是否有搜索关键词、黄金时间、镜头、出片要求、核验来源和放弃条件。

## 状态口径

| status | 含义 |
|---|---|
| planned | 已纳入计划，出发前还要复核开放和到达方式 |
| verified | 已按当轮来源确认可执行 |
| watch | 天气、开放、身体或时间有风险，保留观察 |
| closed | 明确关闭或不可进入 |
| skipped | 主动放弃或被天气/路况砍掉 |
| backup | 替代机位 |

## 现场怎么用

1. 出发前 T-3 把 `data/gannan_photo_spots.csv` 里的 S/A 级机位逐项查一遍。
2. 每个机位至少保留一个地图关键词，优先发给司机确认中文定位。
3. 当天早上把状态改成 `verified/watch/closed`，并在复核日志里补证据。
4. 如果 S 级机位失败，必须按大片交付清单补失败天气替代素材。
5. 天黑、雷雨、高反、闭园或禁入时，放弃条件优先于出片目标。

## 核心覆盖

| 优先级 | 必须覆盖 | 目的 |
|---|---|---|
| S | 扎尕那晨雾/日落石城 | 甘南最重要封面图 |
| S | 莲宝叶则石山湖泊/倒影/蓝调 | 全线最高辨识度主片 |
| A | 拉卜楞寺转经长廊 | 甘南人文秩序 |
| A | 白石崖/甘加草原 | 草原、崖壁和尺度 |
| A | 黄河九曲日落 | 河曲日落大场景 |
