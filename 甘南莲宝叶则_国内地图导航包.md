这份文件面向现场导航。原则是：国内现场优先用高德/百度、司机发的 pin、酒店定位和景区官方指引；Google Maps 只做路线关系参考。不要把搜索结果当成停车点，尤其扎尕那、莲宝叶则、黄河九曲这种景区内交通会变的地方。

## 使用规则

1. 每天晚上先让司机或酒店确认第二天第一个定位点。
2. 搜索结果出现多个同名点时，优先核对县/州：夏河、合作、迭部、若尔盖、阿坝县。
3. 景区内停车、观光车、步行入口和闭园时间，以现场牌示和工作人员为准。
4. 日出/日落机位不要只靠地图，必须问民宿、司机和当天开放范围。
5. 离线前把当天高德/百度路线、酒店地址、门票二维码和司机电话截图保存。

## 容易搜错 / 必须二次确认

| 点位 | 风险 | 处理 |
|---|---|---|
| 扎尕那民宿 | 村内民宿、停车点、景区入口可能不在同一个点 | 让民宿发定位和停车照片，不只搜“扎尕那” |
| 达日/业日机位 | 地图点不一定等于可停车或可进入机位 | 到达后只选一个点，问民宿和司机 |
| 仙女滩/栈道 | 雨后关闭或路滑 | D4 清晨问民宿/景区再决定 |
| 黄河九曲第一湾 | 停车场、扶梯、观景台和酒店不是同一处 | 17:00 前到唐克区域，先问最晚入园和下撤 |
| 莲宝叶则 | 景区入口、观光车/自驾规则、开放范围强时效 | D6 晚、D7 早二次确认，不只按地图导航 |
| 阿坝县酒店 | 同名酒店、老店/新店、停车入口易混 | 下单后让酒店发定位和停车入口照片 |

## 发给司机的定位确认话术

```text
师傅您好，明天我们先按这个顺序走：
起点：____
第一站：____
第二站：____
今日终点：____

请您帮忙确认：
1. 高德/百度这个定位是否准确？
2. 停车点在哪里？
3. 是否有施工、封路、限行或景区交通变化？
4. 如果天气不好，应该先砍哪个点？
5. 最晚几点必须离开，才能不走危险夜路？
```
